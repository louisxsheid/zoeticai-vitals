# zoeticai-vitals

## How to run:

1. Fork this repo
2. `npm install`
3. `npm start`
4. Open with simulator or scan expo code and run on your device

Note: you will need a running mongodb database and a .env file in the root directory with DB_CONNECTION={db_uri}
